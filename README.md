# 🌾 Projeto Agrinho - AgroForte (2026)

Bem-vindo ao repositório oficial do projeto **AgroForte**, um website premium desenvolvido especificamente para o concurso e programa pedagógico **Agrinho**. Este projeto foi desenhado sob os mais altos padrões de design web moderno (HTML5 e CSS3), unindo a estética orgânica do campo com a sofisticação da tecnologia agrícola.

---

## 🚀 O que há no Projeto?

O projeto consiste num website institucional de página única (*Landing Page*), totalmente responsivo, estruturado com semântica limpa e estilizado com técnicas avançadas de CSS. O ecossistema visual foi projetado para transmitir inovação, sustentabilidade e seriedade.

### 🧱 Estrutura de Arquivos
* `index.html`: Arquivo de marcação estrutural utilizando tags semânticas do HTML5 (`<header>`, `<section>`, `<nav>`, `<footer>`).
* `style.css`: Arquivo de estilo contendo variáveis de design, reset moderno, efeitos visuais avançados e regras de responsividade.

### 🎨 Funcionalidades e Destaques Visuais
1.  **Glassmorphism Navbar**: Uma barra de navegação flutuante com efeito de vidro fosco (`backdrop-filter: blur`), que acompanha a rolagem da página de forma elegante.
2.  **Hero Section Impactante**: Uma abertura em ecrã inteiro com imagem de alta definição sobreposta por um gradiente escuro (*overlay*) para garantir legibilidade máxima e contraste tipográfico.
3.  **Grid Assimétrico Adaptativo**: Layout dinâmico que organiza textos e imagens de forma harmoniosa no desktop e empilha-os automaticamente em dispositivos móveis.
4.  **Cards Tridimensionais (Micro-interações)**: Os blocos dos pilares reagem dinamicamente quando o utilizador passa o cursor (*hover*), flutuando e projetando uma sombra suave.
5.  **Design baseado em Variáveis CSS**: Centralização de cores e transições no topo do arquivo CSS (`:root`), permitindo mudar a palete de cores de todo o site em segundos.
6.  **Integração com FontAwesome**: Ícones vetorizados modernos e escaláveis que representam visualmente cada conceito (sustentabilidade, tecnologia e educação).

## 📥 Como Utilizar o Projeto

Para visualizar o site a funcionar no seu computador, siga estes passos simples:

1.  Crie uma pasta no seu computador com o nome `Projeto-Agrinho`.
2.  Dentro dessa pasta, crie um ficheiro de texto chamado **`index.html`** e cole o código HTML fornecido.
3.  Na mesma pasta, crie outro ficheiro de texto chamado **`style.css`** e cole o código CSS fornecido.
4.  Certifique-se de que ambos os ficheiros estão exatamente na mesma pasta.
5.  Dê dois cliques no ficheiro `index.html`. O seu navegador de internet padrão (Chrome, Edge, Firefox, Safari) abrirá o site automaticamente de forma local.

## 🤖 Engenharia de Prompts (Histórico com a IA)

O desenvolvimento deste código de alto nível foi alcançado através de um processo iterativo de refinamento utilizando inteligência artificial. Abaixo está o registo exato e detalhado dos comandos (*prompts*) que utilizou para guiar a IA até ao resultado final.

### 📝 Prompt Inicial (Fase 1: Estruturação Base)
> **Utilizador:** *"faca um site htlm e css sobre agroforte para o agrinho por favor"*

* **Objetivo do Prompt:** Estabelecer o tema principal (Agroforte / Projeto Agrinho) e solicitar as tecnologias de base (HTML e CSS).
* **Ação da IA:** Gerou uma estrutura inicial simplificada, com os arquivos combinados e uma palete de cores básica focada em tons de verde comuns.

### 🚀 Prompt de Refinamento (Fase 2: O Melhor Site do Mundo)
> **Utilizador:** *"o melhor que voce der fazer, separe o html e css. o melhor site do mundo"*

* **Objetivo do Prompt:** Elevar a qualidade técnica ao nível máximo possível (*"o melhor do mundo"*), exigindo práticas profissionais de mercado, separação estrita de responsabilidades (HTML purificado e CSS isolado) e uma interface premium.
* **Ação da IA:** Reestruturou o projeto do zero, introduzindo:
    * Arquitetura CSS moderna com variáveis `:root`.
    * Técnicas de desfoque de fundo (Glassmorphism).
    * Efeitos dinâmicos de transição (`cubic-bezier`).
    * Importação de bibliotecas externas de tipografia e ícones.
    * Responsividade aprimorada (Breakpoints para Mobile e Tablet).

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estrutura declarativa e semântica.
* **CSS3:** Estilização baseada em flexbox, grid, variáveis e animações nativas.
* **FontAwesome (v6.4.0):** Biblioteca externa de ícones em formato vetorial SVG.
* **Google Fonts / Font-Face:** Utilização de fontes do sistema fluidas (`system-ui`, `sans-serif`) para otimização de performance e renderização perfeita em qualquer ecrã.

O **Agrinho** é o maior programa de responsabilidade social do Sistema FAEP, focado em levar educação ambiental e cidadania para as escolas rurais e urbanas.
